# QuickBite - Software Requirements Specification (SRS)

## Overview

This directory contains the complete Software Requirements Specification (SRS) for the **QuickBite - Food Delivery Platform**.

The document follows a structured Requirement Engineering approach inspired by IEEE SRS guidelines and is organized into multiple sections covering functional, non-functional, interface, and system requirements.

---

## Document Structure

| Section                               | Description                                                                  |
| ------------------------------------- | ---------------------------------------------------------------------------- |
| 01_Introduction.md                    | Purpose, Scope, Definitions, References, Document Overview                   |
| 02_Overall_Description.md             | Product Perspective, User Classes, Assumptions, Constraints, Dependencies    |
| 03_System_Features.md                 | High-level overview of all functional modules                                |
| 04_Non_Functional_Requirements.md     | Performance, Scalability, Availability, Reliability, Security, Privacy       |
| 05_External_Interface_Requirements.md | User, Software, Hardware and Communication Interfaces                        |
| 06_Other_Requirements.md              | Business Rules, Compliance, Logging, Auditing and Miscellaneous Requirements |
| 07_Appendix.md                        | Glossary, References and Supporting Information                              |
